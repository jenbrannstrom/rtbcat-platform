## Issue 2: Creative Preview 500 Error
- **Endpoint**: `GET /api/creatives/207573285_274443121_banner_intertpl_9259`

### Root Cause
```
TypeError: Creative.__init__() got an unexpected keyword argument 'first_import_batch_id'
```

The database `creatives` table has column `first_import_batch_id` (added in schema) but the `Creative` model in `storage/models.py` was missing this field.

### Fix Applied
Added missing field to `storage/models.py`:
```python
first_import_batch_id: Optional[str] = None  # Import batch ID when first seen
```
